floosie

Universal stream processing platform. Pipe anything to anything. Built on sflow with a 98-type chunk model, byte-level utilities, observability, WebSocket and ACP bridging, and a CLI.

Repository · npm

Install

npm install floosie

Quick start

import { createProcessor, json, text } from "floosie";

const upper = createProcessor({
  name: "upper",
  input: (async function* () {
    yield json({ msg: "hello" });
    yield json({ msg: "world" });
  })(),
  transform: (flow) => flow.map((c) => text(String(c.data.msg).toUpperCase())),
});

for await (const c of upper.output) console.log(c.data);

CLI

floosie pipe --in file:data.json --out ws://localhost:7878
floosie bridge --port 7878
floosie mux --urls ws://a:7878,ws://b:7878 --strategy round-robin
floosie inspect data.bin

Pillars

  • Chunks — 98 typed payloads with codecs, MIME auto-detection, structured/binary/scalar/embedding families.
  • Processors — composable transforms with lifecycle, registry, stdout/stderr split, ACP bridge.
  • Operators — mux, split, gate, scan, zip, batch, window, throttle, debounce, take, drop, distinct, parallel, withBackpressure.
  • Buffer utilities — hex/base64/base32, hash, hmac, gzip/brotli, hexdump, rechunk, splitOn.
  • Observability — debugSnapshot(), debugLog(), ring-buffered logs, registry transitions.
  • WebSocket — wsSource, wsSink, wsBridge, muxWsClients with round-robin / broadcast / hash strategies.

For the full API surface, see the README on GitHub.
